# kelly-lists

Word lists in the following languages:

- Arabic
- Chinese
- English
- Italian
- Norwegian
- Russian
- Swedish

## usage

```javascript
const { swedish } = require(`kelly-lists`)
console.log(swedish[2])
// bakre
```
